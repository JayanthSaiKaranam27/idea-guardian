# Idea Guardian

## Theme
AI at Work — Productivity & Teamwork Reimagined

## Project Description
Idea Guardian is an AI‑powered application that transforms messy team chat transcripts into structured insights.  
It automatically:
- 💡 Extracts **ideas** into an Ideas Vault
- ✅ Captures **tasks** as Action Items
- 🎭 Analyzes **team mood** with sentiment scoring
- 📊 Visualizes mood trends with a Sentiment Timeline Chart
- 🏷️ Tags ideas with categories (Design, Automation, Workflow, Team Morale)
- 📥 Exports results to CSV for easy sharing

This ensures no idea gets lost, no task falls through the cracks, and the team’s emotional state is visible in real time.
